## 作品一：《王者荣耀》全系统竞品拆解报告
文档路径：`/01_KingHonor_Analysis/王者荣耀全系统竞品拆解报告2026.05.pdf`
### 项目概述
从策划底层视角完整拆解成熟MOBA产品全模块，逆向推导成熟竞技游戏留存、付费、数值平衡底层逻辑，定位产品现存痛点并输出可落地优化方案，沉淀通用竞技游戏设计方法论。
### 核心工作内容
1. 全体系拆解：战斗对局、英雄成长、局内/局外双经济、分层付费、社交匹配五大核心系统完整梳理
2. 数值逆向分析：拆解英雄强度曲线、经济产出消耗循环、胜率匹配底层逻辑，定位英雄数值失衡、匹配体验割裂痛点
3. 落地优化输出：针对匹配机制、新手留存、付费体感三大问题给出完整迭代方案
4. 配套产出：玩家生命周期循环图、全链路货币流转流程图

本报告仅为个人行业学习、求职作品集使用，不存在商用行为；文中所有机制、数值均基于玩家体验归纳推导，未获取、泄露厂商任何内部保密配置、原始源码、官方原画素材。

## 作品二：原创轻量化MOBA《星途对决》完整GDD
文档路径：`/02_StarDuel_MOBA_GDD/星途对决完整GDD2026.06.pdf`
### 项目背景
针对传统MOBA两大行业痛点：新手上手门槛过高、单局对局时长过长，自主设计8-12分钟轻量化5V5竞技产品，完整产出从核心玩法到长线运营的全套策划文档。
### 个人独立完成全部工作
1. 核心玩法创新：缩短对局节奏，简化复杂操作门槛，兼顾轻度玩家与竞技用户体验
2. 英雄与数值体系：从零设计全套英雄技能机制、成长属性曲线、对局经济平衡数值表
3. 原创养成系统：无抽奖氪金的星纹养成体系，消除抽卡焦虑，付费仅开放外观皮肤，主打公平竞技
4. 经济闭环推演：结合Python、运筹学、ERP库存流转思路，搭建全局货币产出消耗模型，规避通胀/通缩问题(暂未完成)
5. 长线运营规划：完整版本更新节奏、活动模板、用户分层留存方案
6. 配套产出：对局流程、养成链路、货币循环、付费转化多类可视化流程图(暂未完成)
